柯 P 滑進來對你說政見

Mr. KP slide in and talks to you

簡介

本 Plugin 之內容與資訊來源，使用「柯文哲野生官網 unlimited」所提供之 API 服務
讓您可以簡單、輕鬆地在 個人網站 或 網誌 中埋入「柯P彩蛋」，並讓更多人得以了解柯文哲之政見
更多相關說明，請參考 柯文哲野生官網 unlimited


大安連先生：甜甜圈 ... 在安裝「柯 P 滑進來對你說政見」之後 ... 我震驚甜甜圈，了

當您在的網頁中安裝了「柯 P 滑進來對你說政見」Plugin 之後，神奇的事情發生了
到訪者進入您的頁面、並捲動到頁面底端之後，柯 P 就會 滑 出 來 向 他 們 說 明 政 見
每次的瀏覽，Plugin 都會從 柯文哲官方網站 中隨機的取得一筆政見，來對瀏覽者說明。

如何安裝

步驟 1 : 請先在您的 網誌 或 個人網頁 中，引用 jquery.js

<script src="http://code.jquery.com/jquery-1.11.1.min.js"></script>

步驟 2 : 請在您的網頁中，引用 kptalks.js

<script src="http://goooooooogle.github.io/kp/kptalks.js"></script>
